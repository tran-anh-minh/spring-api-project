## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-10 | Injection | app.py:ask | mitigate | Question goes through pipeline which validates generated SQL via sqlglot qualify(). Execution gated by explicit execute=True parameter. Executor rejects non-SELECT. |
| T-04-11 | Injection | app.py:define_metric | mitigate | SQL fragment validated by sqlglot.parse_one() and keyword blacklist in resolver.define_metric() before storage. |
| T-04-12 | Information Disclosure | app.py:coverage,data_quality | accept | These tools intentionally expose knowledge store statistics. No PII in knowledge store -- only schema metadata. Local-only MCP server. |
| T-04-13 | Denial of Service | app.py:impact,forensics | mitigate | max_depth parameter limits BFS traversal. Default 3/5 with upper bound check. bfs_graph already has cycle detection. |
</threat_model>
