```python
from pydantic import BaseModel

class ColumnInfo(BaseModel):
    column_name: str
    data_type: str | None = None
    is_nullable: bool = True
    is_primary_key: bool = False
    is_unique: bool = False
    default_value: str | None = None
    ordinal_position: int | None = None

class ConstraintInfo(BaseModel):
    constraint_type: str  # "PRIMARY KEY", "FOREIGN KEY", "UNIQUE"
    columns: list[str]
    ref_table: str | None = None  # FK target table
    ref_columns: list[str] | None = None  # FK target columns

class IndexInfo(BaseModel):
    index_name: str
    table_name: str
    schema_name: str | None = None
    is_unique: bool = False
    columns: list[str]

class TableInfo(BaseModel):
    table_name: str
    schema_name: str | None = None
    columns: list[ColumnInfo]
    constraints: list[ConstraintInfo] = []

class RelationshipInfo(BaseModel):
    source_table: str
    target_table: str
    relationship_type: str  # "fk_declared"
    source_column: str | None = None
    target_column: str | None = None
    confidence: float = 1.0

class ParseResult(BaseModel):
    tables: list[TableInfo] = []
    indexes: list[IndexInfo] = []
    relationships: list[RelationshipInfo] = []
    warnings: list[str] = []
```

Create `db_wiki/ingest/ddl_parser.py` with these functions:

1. `parse_ddl_file(sql_text: str) -> tuple[list[exp.Expression], list[str]]`:
   - Calls `sqlglot.parse(sql_text, dialect="tsql")`
   - Filters out `None` results and `exp.Command` nodes (sqlglot fallback for unparseable syntax — per D-04)
   - Logs warnings for skipped statements via `logging.getLogger(__name__)`
   - Returns (valid_statements, warning_messages)

2. `extract_create_table(stmt: exp.Create) -> TableInfo`:
   - Gets table_name from `stmt.this.name` (the Schema/Table node)
   - Gets schema_name from `stmt.this.db` if present
   - Iterates `stmt.find_all(exp.ColumnDef)` for columns:
     - `col_def.name` for column name
     - `col_def.kind.sql(dialect="tsql")` for data type
     - Check `col_def.args.get("constraints", [])` — for each constraint, check `constraint.kind`:
       - `isinstance(constraint.kind, exp.PrimaryKeyColumnConstraint)` -> is_primary_key=True
       - `isinstance(constraint.kind, exp.NotNullColumnConstraint)` -> is_nullable=False
       - `isinstance(constraint.kind, exp.UniqueColumnConstraint)` -> is_unique=True
       - `isinstance(constraint.kind, exp.DefaultColumnConstraint)` -> default_value=constraint.kind.this.sql()
   - Iterates `stmt.find_all(exp.PrimaryKey)` for table-level PK constraints
   - Iterates `stmt.find_all(exp.ForeignKey)` for table-level FK constraints — extract ref_table and ref_columns from `fk.args.get("reference")`
   - Returns `TableInfo(...)` with columns and constraints

3. `extract_create_index(stmt: exp.Create) -> IndexInfo | None`:
   - Checks `stmt.args.get("kind")` equals "INDEX"
   - Gets index_name from the index expression node
   - Gets table_name from `stmt.this.name` or similar AST path
   - Gets is_unique from `stmt.args.get("unique")` 
   - Gets columns from the index column expressions
   - Returns `IndexInfo(...)` or `None` if not an index statement

4. `extract_alter_table_constraint(stmt: exp.AlterTable) -> list[RelationshipInfo]`:
   - Finds FK constraints added via ALTER TABLE
   - For each FK: extracts source_table, target_table, source_column, target_column
   - Returns `RelationshipInfo(relationship_type="fk_declared", ...)`

5. `parse_ddl(sql_text: str) -> ParseResult`:
   - Calls parse_ddl_file to get valid statements
   - Routes each to extract_create_table, extract_create_index, or extract_alter_table_constraint based on type
   - Aggregates into `ParseResult(tables=..., indexes=..., relationships=..., warnings=...)`
   - Also extracts FK relationships from inline table-level constraints and adds them to relationships

6. `ingest_ddl(conn: sqlite3.Connection, parse_result: ParseResult) -> dict`:
   - For each TableInfo: INSERT into db_tables with current UTC timestamps (both ISO and epoch for all 4 temporal concepts per D-02), valid_until=NULL, invalidated_at=NULL
   - For each ColumnInfo within a table: INSERT into db_columns with FK to parent table id
   - For each RelationshipInfo: INSERT into db_relationships with type="fk_declared"
   - For each IndexInfo: INSERT into db_indexes with FK to parent table id, columns_json as JSON array
   - Returns `{"tables": count, "columns": count, "relationships": count, "indexes": count}`
   - Uses `datetime.now(timezone.utc)` for timestamps and `int(datetime.now(timezone.utc).timestamp())` for epoch values

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Denial of Service | ddl_parser.py parse_ddl_file | mitigate | Check file size against config.ingest.max_file_size_mb (default 50MB) before passing to sqlglot. Reject with clear error if exceeded. |
| T-02-02 | Tampering | ddl_parser.py ingest_ddl | mitigate | Use parameterized SQL (?) placeholders for all INSERT statements — never string-format user data into SQL. SQLite parameterized queries prevent injection. |
| T-02-03 | Tampering | ddl_parser.py parse_ddl_file | accept | sqlglot parses SQL structurally into AST — no SQL is ever executed against any database. Malicious SQL in DDL files is harmless (parsed, not run). |
