```python
class MutationInfo(BaseModel):
    table_name: str
    mutation_type: str  # "insert", "update", "delete", "merge"
    columns: list[str] = []

class BranchInfo(BaseModel):
    branch_index: int
    condition_text: str | None = None
    branch_type: str  # "if", "else", "case_when", "while"
    tables_touched: list[str] = []
    nesting_depth: int = 0

class CallChainInfo(BaseModel):
    callee_name: str
    callee_schema: str | None = None
    is_dynamic: bool = False

class EnumDetection(BaseModel):
    table_name: str
    column_name: str
    values: list[dict] = []
    confidence: float = 0.5
    detection_method: str

class StateTransitionInfo(BaseModel):
    table_name: str
    column_name: str
    from_value: str
    to_value: str
    confidence: float = 0.9

class SPInfo(BaseModel):
    procedure_name: str
    schema_name: str | None = None
    body_hash: str = ""
    table_refs: list[str] = []
    mutations: list[MutationInfo] = []
    branches: list[BranchInfo] = []
    call_chains: list[CallChainInfo] = []
    enum_detections: list[EnumDetection] = []
    state_transitions: list[StateTransitionInfo] = []
    parse_quality: float = 1.0
    is_degraded: bool = False
    has_dynamic_sql: bool = False
    partial_ast: bool = False
    dynamic_sql_locations: list[dict] = []
    warnings: list[str] = []

class SPParseResult(BaseModel):
    procedures: list[SPInfo] = []
    relationships: list[RelationshipInfo] = []
    warnings: list[str] = []
```

**Imports:**
```python
import hashlib
import json

import sqlglot

from db_wiki.core.models import (

    RelationshipInfo, SPInfo, SPParseResult, StateTransitionInfo,
)
```

**Functions:**

1. `compute_body_hash(sql_text: str) -> str` — normalize whitespace with `" ".join(sql_text.split())`, return SHA-256 hex digest.

2. `parse_sp_file(sql_text: str) -> tuple[list[exp.Expression], list[str]]` — Same ErrorLevel.WARN pattern as ddl_parser. Filter for `exp.Create` with `kind == "PROCEDURE"`. Skip Command nodes and non-SP statements with warnings.

3. `extract_table_refs(sp_stmt: exp.Create) -> tuple[list[str], bool]` — Two-pass extraction per RESEARCH.md Pattern 2:
   - Pass 1: `sp_stmt.find_all(exp.Table)` — collect table names, excluding the SP name itself
   - Pass 2: `sp_stmt.find_all(exp.Command)` — regex `r'\b(?:FROM|JOIN|UPDATE|INTO)\s+([\w\.]+)'` on `.sql()` output, strip schema prefix
   - Return (table_names, has_command_nodes)

4. `extract_mutations(sp_stmt: exp.Create) -> list[MutationInfo]` — Find all exp.Insert (target via `.this.this.name`), exp.Update (`.this.name`), exp.Delete (`.this.name`), exp.Merge (`.this.name`). For INSERT/UPDATE, collect column names from expressions.

5. `extract_branches(sp_stmt: exp.Create) -> list[BranchInfo]` — Find exp.If nodes (branch_type="if", condition from `.this`), exp.Command nodes with ELSE text (branch_type="else"), CASE/WHEN via exp.Case (branch_type="case_when"). Track nesting depth via parent chain. Extract tables_touched per branch via find_all(exp.Table) scoped to each branch node.

6. `classify_execute(exe: exp.Execute) -> dict` — Per RESEARCH.md Pattern 4: check for dynamic SQL (Subquery/Paren in `.this`), sp_executesql (name match), or static call (Table node with `.name`).

7. `extract_call_chains(sp_stmt: exp.Create) -> tuple[list[CallChainInfo], list[dict]]` — For each `sp_stmt.find_all(exp.Execute)`, classify and build CallChainInfo. Return (call_chains, dynamic_sql_locations).

8. `extract_enum_detections(sp_stmt: exp.Create) -> list[EnumDetection]` — Per D-04: Find exp.Case nodes. For each case, iterate `.args["ifs"]` (exp.If nodes). Extract column from `.this` (the condition), literal value and THEN label. Build EnumDetection with detection_method="case_when", confidence=0.7.

9. `extract_state_transitions(sp_stmt: exp.Create) -> list[StateTransitionInfo]` — Per RESEARCH.md Pattern 5: Find exp.Update, match SET col=literal WHERE col=literal on same column. Confidence 0.9 per D-15.

10. `compute_parse_quality(sp_stmt: exp.Create) -> tuple[float, bool]` — Per RESEARCH.md Pattern 3: count Anonymous nodes / total nodes. Degraded if ratio > 0.05.

11. `extract_sp_info(sp_stmt: exp.Create, sql_text: str) -> SPInfo` — Orchestrate all extraction functions. Build SPInfo with all fields populated.

12. `parse_sp(sql_text: str) -> SPParseResult` — Parse file, extract SPInfo for each SP, build relationships (reads_from for SELECT refs, writes_to for mutation targets, joins_with for JOIN tables). Return SPParseResult.

13. `invalidate_procedure(conn: sqlite3.Connection, procedure_id: int, now_iso: str, now_ts: int) -> None` — Per RESEARCH.md Pattern 10: invalidate db_procedures row and cascade to sp_branches, sp_reliability, sp_call_chains, enum_values, state_transitions, bitmask_definitions, column_aliases.

14. `ingest_sp(conn: sqlite3.Connection, sp_result: SPParseResult) -> dict[str, int]` — Write to DB following ingest_ddl pattern:
    - For each SPInfo: check existing procedure by name via `current_db_procedures`. If exists and body_hash differs, call invalidate_procedure on old row. If exists and body_hash matches, skip (no re-parse needed per D-11).
    - INSERT into db_procedures with body_hash
    - INSERT into sp_reliability (parse_quality, is_degraded, has_dynamic_sql, partial_ast, dynamic_sql_locations_json)
    - INSERT into sp_branches for each branch
    - INSERT into sp_call_chains for each call chain
    - INSERT into enum_values for each enum detection
    - INSERT into state_transitions for each state transition
    - INSERT into db_relationships for reads_from/writes_to relationships (source_id = procedure row id, target_id = lookup table by name from current_db_tables)
    - All INSERTs use ? parameterized placeholders (T-02-02)
    - try/commit/except/rollback pattern

15. `detect_content_type(sql_text: str) -> str` — Per D-13: Parse with sqlglot, check if any statement is `exp.Create` with `kind == "PROCEDURE"`. Return "sp" if yes, "ddl" if contains CREATE TABLE/INDEX/ALTER, "unknown" otherwise.
